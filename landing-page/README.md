# ChegouAqui Landing Page

Landing page standalone para vender o serviço ChegouAqui.

## Estrutura
- index.html - Landing page completa com SEO otimizado
- css/styles.css - Estilos com identidade visual da marca
- js/script.js - Animações e formulário de leads
- images/ - Imagens otimizadas

## Deploy
Esta landing page pode ser servida como arquivos estáticos ou integrada ao app principal.

## API Backend
O formulário envia dados para: POST /api/leads
